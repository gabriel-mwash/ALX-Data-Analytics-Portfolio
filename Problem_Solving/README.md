# 📝 Problem-Focused Analysis Project

---

## Overview

This project explores how to **define a problem**, craft a **concise and clear problem statement**, ask the right questions to determine clarity, and identify **best approaches** to address the problem.  

> 💡 **Goal:** Demonstrate how **complex problems can be examined systematically** before attempting to solve them.

---

## 🎯 Project Purpose

The purpose of this project is to demonstrate an understanding of:

- **What constitutes a problem** in a real-world context  
- **How to formulate a clear and concise problem statement**  
- **How to ask structured questions** to clarify a problem  
- **How different frameworks** can be used to understand and approach a problem  

> ⚡ **Focus:** Problem exploration and analysis rather than immediate implementation of solutions.

---

## 🌍 Problem Context
  
> Urban areas in Kenya, especially Nairobi and Mombasa, face **severe plastic and solid waste pollution**. Streets and rivers are clogged, leading to **flooding, environmental degradation, and public health risks**.

**Analysis Focus:**

- Understanding the problem in its **real-life context**  
- Identifying the **key contributing factors**  
- Examining **possible approaches** to address it  

---

## ❓ Key Questions (5W2H Framework)

The analysis uses **structured questioning** to clarify the problem:

| Question | Purpose |
|----------|---------|
| **What** exactly is the problem? | Identify the core issue |
| **Why** is it a problem? | Understand its impact |
| **Where** does the problem occur? | Locate affected areas |
| **Who** is affected? | Identify stakeholders |
| **When** does the problem occur? | Determine timing/frequency |
| **How** does the problem manifest? | Observe symptoms or signs |
| **How much** of a problem is it? | Measure scope and scale |

> 💬 Using a structured approach ensures **all dimensions of the problem are captured** before attempting solutions.

---

## 🛠 Analytical Approaches Used

### 1️⃣ Design Thinking

> A **human-centered** and collaborative approach to realize **new solutions** for **ill-structured and complex problems**.

### 2️⃣ Pragmatic Thinking
  
> A **logical and structured approach** that breaks down problems or tasks into **smaller parts**, solving each part systematically.
